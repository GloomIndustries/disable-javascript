About Disable JavaScript (2.2.1)

First off, thank you for using this web extension!

If you find any bugs or have any suggestions/ideas for this web extension, please create a new issue on the projects GitHub page.
I've received a few reports where this web extension is not always working. However, I was not able to reproduce this bug as I didn't get enough information to do so.

If you happen to find this web extension not working on a particular machine/browser/website, please provide me details like browser, browser version, operating system and website url so I can check the web extension for any bugs. Read more about this on GitHub.

Thank you for your support!

What's new (2.2.1)

Fixed a caching bug related to <noscript>-tags when disabling/enabling JS.

2.2.0

There was a bug on Firefox Quantum which didn't update web extension icons after installation. As a workaround for this, this web extension used to force refresh all active tabs on installation/update. After some feedback on GitHub and in form of reviews, I'm glad to tell you that this behavior was removed since the Firefox bug was fixed in the meantime.
In addition to this, the about page now also only pops up when there was a major or minor web extension update.
Browser updates or patch updates of the web extension won't open the about page any longer!

You can now add base domains that automatically disable JS for any sub domains when adding a new domain via the settings page.
Be careful when using this feature though, as complete inheritance is not supported yet.

Shortcuts

  • Alt + Shift + 1
    Toggle JavaScript state
  • Alt + Shift + 2
    Open Disable JavaScript settings
  • Important
    You can enable or disable shortcuts in the settings page.
