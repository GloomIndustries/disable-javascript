About Disable JavaScript

First off, thank you for using this web extension!

If you find any bugs or have any suggestions/ideas for this web extension, please create a new issue on the projects GitHub page.
I've received a few reports where this web extension is not always working. However, I was not able to reproduce this bug as I didn't get enough information to do so.

If you happen to find this web extension not working on a particular machine/browser/website, please provide me details like browser, browser version, operating system and website url so I can check the web extension for any bugs.

Thank you for your support!

Shortcuts

  • Alt + 1
    Toggle JavaScript state
  • Alt + 2
    Open Disable JavaScript settings
